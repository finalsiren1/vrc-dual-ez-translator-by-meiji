# Shortcut Keys (Hotkeys)

SyncVRC supports **Global Hotkeys**, meaning you can control the translation systems without having to Alt-Tab out of VRChat or bring the application window to the foreground. 

You can configure these hotkeys at the bottom of the **Settings** tab. If you prefer to click the buttons manually with your mouse, you can set any of these to `None`.

## Available Hotkey Actions

### 1. Start Outgoing (Default: F2)
* **Function:** Toggles the Outgoing translation system On and Off. 
* **Usage:** Press the key once to start listening to your microphone. Press it again to stop.

### 2. Start Incoming (Default: F3)
* **Function:** Toggles the Incoming translation system On and Off.
* **Usage:** Press the key once to start listening to other players. Press it again to stop.

### 3. Hold to Speak / Push Mode (Default: None)
* **Function:** Acts as a Push-to-Talk button for your translations.
* **Important Requirement:** This hotkey **only works** if you have selected the **Push to Record** mode in the Translation tab AND the Outgoing system is actively running.
* **Usage:** Press and **hold** the key while you are speaking. Release the key when you finish your sentence to instantly process the audio and send the translation to VRChat.

## Supported Keys

To prevent conflicts with standard VRChat controls (such as `WASD` for movement or `V` for the in-game microphone), SyncVRC currently restricts hotkeys to the following safe options:

* `F1` through `F12`
* `Insert`, `Delete`, `Home`, `End`, `PageUp`, `PageDown`, `Pause`
