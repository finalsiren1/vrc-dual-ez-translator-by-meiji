# Obtaining a DeepL API Key

SyncVRC supports translation via the **DeepL API**. This service is well-known for its natural-sounding translations, particularly for European and Asian languages.

:::warning Disclaimer
This tutorial may be incomplete or slightly outdated as the developer does not primarily use the DeepL API for daily testing. Please refer to the official DeepL documentation if you encounter any discrepancies.
:::

---

## Steps to Obtain the Key

1.  **Create a DeepL Account:** Go to the [DeepL Pro](https://www.deepl.com/pro-api) website.
2.  **Select the API Plan:** Choose the **"DeepL API Free"** (up to 500,000 characters/month) or **"DeepL API Pro"** plan. 
    * *Note: Even for the Free plan, DeepL usually requires a valid credit card for identity verification.*
3.  **Access the Account Console:** Once logged in, navigate to the **"Account"** tab.
4.  **Find your Authentication Key:** Scroll down to the bottom of the account page to find your **"Authentication Key for DeepL API"**.
5.  **Copy and Paste:** Copy the key and paste it into the **Settings** tab in SyncVRC under the DeepL API section.

---

## DeepL API Free vs. Pro

* **DeepL API Free:** Provides a generous limit of characters per month for free, but requires credit card verification in most regions.
* **DeepL API Pro:** Removes character limits (pay-as-you-go) and offers maximum data security, ensuring your texts are never stored or used to train their models.
