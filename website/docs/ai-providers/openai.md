# Obtaining an OpenAI API Key

SyncVRC supports translation using **OpenAI's models** (such as GPT-4o or GPT-3.5). OpenAI is a popular choice known for its high-quality translation and broad language support.

:::warning Disclaimer
This tutorial may be incomplete or slightly outdated as the developer does not primarily use the OpenAI API for daily testing. Please refer to the official OpenAI documentation for the most accurate information.
:::

---

## Steps to Obtain the Key

1.  **Create an OpenAI Account:** Visit the [OpenAI Platform](https://platform.openai.com/) and sign up or log in.
2.  **Access API Keys:** Navigate to the **"API Keys"** section in the left sidebar (under the "Dashboard" or "Settings" menu).
3.  **Create Secret Key:** Click on the **"+ Create new secret key"** button.
4.  **Name and Copy:** Give your key a name (e.g., "SyncVRC") and click **Create**. **Copy the key immediately**, as you will not be able to see it again once you close the window.
5.  **Configure in SyncVRC:** Open the **Settings** tab in SyncVRC, select `OpenAI` as your engine, and paste the key.

---

## Important: Usage Credits

* **Prepaid System:** OpenAI typically requires you to purchase "credits" in advance before the API will function. Even with a new account, you may need to add at least $5 to your balance to start using the translation service.
* **Tiers:** Your "usage tier" (which determines your rate limits) will increase as you spend more on the platform over time.
