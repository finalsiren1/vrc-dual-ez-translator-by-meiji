# OpenAI APIキーの取得

SyncVRCは、**OpenAIのモデル**（GPT-4oやGPT-3.5など）を使用した翻訳をサポートしています。OpenAIは、高品質な翻訳と幅広い言語サポートで知られる人気のある選択肢です。

:::warning 免責事項
開発者は日常的なテストにOpenAI APIを主に使用していないため、このチュートリアルは不完全であるか、わずかに古い可能性があります。最も正確な情報については、公式のOpenAIドキュメントを参照してください。
:::

---

## キーを取得する手順

1. **OpenAIアカウントの作成:** [OpenAI Platform](https://platform.openai.com/)にアクセスし、サインアップまたはログインします。
2. **APIキーへのアクセス:** 左側のサイドバー（「Dashboard」または「Settings」メニューの下）の**"API Keys"**セクションに移動します。
3. **シークレットキーの作成:** **"+ Create new secret key"**ボタンをクリックします。
4. **名前を付けてコピー:** キーに名前（「SyncVRC」など）を付け、**Create**をクリックします。ウィンドウを閉じると再び表示できなくなるため、**すぐにキーをコピーしてください**。
5. **SyncVRCでの設定:** SyncVRCの**Settings（設定）**タブを開き、エンジンとして`OpenAI`を選択し、キーを貼り付けます。

---

## 重要: 使用量クレジット

* **プリペイドシステム:** OpenAIは通常、APIが機能する前に「クレジット」を事前に購入する必要があります。新しいアカウントであっても、翻訳サービスの使用を開始するには、残高に少なくとも5ドルを追加する必要がある場合があります。
* **階層:** プラットフォームでより多くを費やすにつれて、時間とともに「使用量階層」（レート制限を決定する）が上がります。
