# DeepL APIキーの取得

SyncVRCは**DeepL API**による翻訳をサポートしています。このサービスは、特にヨーロッパとアジアの言語において、自然に聞こえる翻訳でよく知られています。

:::warning 免責事項
開発者は日常的なテストにDeepL APIを主に使用していないため、このチュートリアルは不完全であるか、わずかに古い可能性があります。矛盾がある場合は、公式のDeepLドキュメントを参照してください。
:::

---

## キーを取得する手順

1. **DeepLアカウントの作成:** [DeepL Pro](https://www.deepl.com/pro-api)のウェブサイトにアクセスします。
2. **APIプランの選択:** **"DeepL API Free"**（月間最大500,000文字）または**"DeepL API Pro"**プランを選択します。
    * *注意: 無料プランであっても、DeepLは通常、本人確認のために有効なクレジットカードを必要とします。*
3. **アカウントコンソールへのアクセス:** ログインしたら、**"アカウント"**タブに移動します。
4. **認証キーを見つける:** アカウントページの一番下までスクロールして、**"DeepL APIの認証キー"**を見つけます。
5. **コピーして貼り付け:** キーをコピーして、SyncVRCの**Settings（設定）**タブ内のDeepL APIセクションに貼り付けます。

---

## DeepL API Free vs. Pro

* **DeepL API Free:** 無料で月間の十分な文字数制限を提供しますが、ほとんどの地域でクレジットカードによる確認が必要です。
* **DeepL API Pro:** 文字数制限がなくなり（従量課金制）、最大限のデータセキュリティを提供し、テキストが保存されたりモデルのトレーニングに使用されたりしないことを保証します。
