# トラブルシューティングと一般的な問題

SyncVRCの使用中に問題が発生した場合は、バグを報告する前に、このガイドで最も一般的な問題とその解決策を確認してください。

## 1. AI処理とハードウェアのエラー

**エラー:** `CUDA/GPU Error: No compatible NVIDIA GPU found.` またはマイクの起動時にアプリがクラッシュする。
* **原因:** アプリはGPUを使用してFaster-Whisperモデルを処理しようとしていますが、必要なNVIDIAドライバーが見つかりません。
* **解決策:**
  * **NVIDIA GPU**をお持ちの場合は、[NVIDIA CUDA Toolkit 12.x](./tutorial/initial-setup)をインストールする必要があります。インストール後にPCを再起動してください。
  * **AMD、Intel、または専用GPUがない**場合は、**Settings（設定）**タブに移動し、**AI Processing Device（AI処理デバイス）**を`CPU (Fallback)`に変更してください。

## 2. APIと翻訳のエラー

**エラー:** `API Limit Exceeded: Too many requests.`
* **原因:** 無料枠のAPIキー（特にGoogle Gemini）を使用しており、話す量が多すぎたり速すぎたりして、レート制限に達しました。
* **解決策:** 話すのを1分間やめて、制限がリセットされるのを待ちます。恒久的な解決策としては、APIアカウントを従量課金プランにアップグレードしてください。または、SyncVRCのモードを*Auto (Continuous)*から*Push to Record*に切り替えてください。

**エラー:** `API Error: Invalid API Key.`
* **原因:** APIキーが空であるか、正しくコピーされていないか、クラウドプロバイダーのアカウントが制限されています。
* **解決策:** **Settings（設定）**タブでAPIキーを再確認してください。キーの先頭または末尾に余分なスペースがないことを確認してください。

**エラー:** `API Request Error: 400 Bad Request`（または類似の400エラー）
* **原因:** これは通常、プロバイダーによってAIモデル名が変更された場合、または送信されたテキストがクラウドプロバイダーの安全ガイドライン（例：極端な冒涜）に違反している場合に発生します。
* **解決策:** SyncVRCでは安全フィルターはデフォルトで無効になっていますが、プロバイダーは特定の単語をハードブロックする場合があります。別の文を話してみてください。

## 3. VRChat OSC接続の問題

**問題:** OSCステータスが`🔴 Not Detected`と表示される、または翻訳がVRChatに表示されない。
* **原因:** VRChatがデフォルトのポート（`9000`）でOSCデータを送受信していません。
* **解決策:**
  1. VRChatのラジアルメニューを開きます。
  2. **Options > OSC**に進み、**Off**に切り替えてから、**On**に戻して接続をリセットします。
  3. 他のOSCアプリケーション（フェイストラッカーや心拍数モニターなど）がポート`9000`または`9001`をブロックしていないことを確認します。

## 4. オーディオルーティングとマイクの問題

**問題:** 受信システムが自分自身の声、またはYouTubeの動画を文字起こししている。
* **原因:** *Speaker Loopback（スピーカーループバック）*設定で物理的なスピーカーまたはヘッドフォンを選択しています。
* **解決策:** 仮想オーディオケーブル（VB-Cableなど）またはWindowsステレオミキサーを使用する必要があります。VRChatのオーディオ**のみ**をこの仮想ケーブルにルーティングし、SyncVRCの受信設定でその仮想ケーブルを選択してください。

**問題:** アプリには「Listening...」と表示されるが、声が翻訳されない。
* **原因:** AIがあなたの声を聞き取れないか、マイクの音量が低すぎます。
* **解決策:**
  * マイクがハードウェアでミュートされていないことを確認してください。
  * Windowsのサウンド設定を確認し、マイクがデフォルトの入力デバイスであり、入力音量が十分に高いことを確認してください。
  * 高度なオーディオ設定（Advanced Audio Settings）で**Silence Timeout（無音タイムアウト）**を調整してください。高すぎると、AIが処理するまでに長く待ちすぎる可能性があります。

## 5. ミュート同期が機能しない
**問題:** VRChatでマイクをミュートしたが、SyncVRCが翻訳を続けている。
* **原因:** OSCが有効になっていないか、Mute Sync（ミュート同期）ボックスにチェックを入れていません。
* **解決策:** **Audio Setup（オーディオ設定）**に進み、**Enable VRC Mute Sync（VRCミュート同期を有効にする）**にチェックが入っていることを確認してください。また、左下のOSCステータスが緑色（`🟢 Detected`）であることを確認してください。

## まだヘルプが必要ですか？

問題がここにリストされていない場合は、設定（Settings）タブで**Telemetry (Crash Reports)（テレメトリ（クラッシュレポート））**が有効になっていることを確認してください。これにより、アプリは開発者にエラーログを安全に送信できます。

また、[GitHub Issuesページ](https://github.com/finalsiren1/SyncVRC/issues)でサポートを求めたり、バグを報告したりすることもできます。
