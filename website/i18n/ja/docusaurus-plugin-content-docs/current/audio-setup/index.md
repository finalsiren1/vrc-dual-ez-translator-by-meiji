# 🎧 オーディオ設定とルーティング

オーディオ設定ガイドへようこそ。オーディオを適切にルーティングすることは、SyncVRCがあなたや他のプレイヤーの声をはっきりと「聞く」ことを保証するための最も重要なステップです。

このセクションでは、送信（Outgoing）翻訳用の物理的なマイクと、受信（Incoming）翻訳用の仮想オーディオデバイスの両方を設定する方法について説明します。

左側のサイドバーから特定のガイドを選択するか、**「Next（次へ）」**をクリックして送信マイクの設定を開始してください。
