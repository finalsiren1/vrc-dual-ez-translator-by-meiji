# ⚙️ 高度なオーディオ設定

これらのスライダーを使用すると、AIが音声を処理する方法を微調整でき、速度、精度、個人の話すペースのバランスを取るのに役立ちます。

## Silence Timeout（無音タイムアウト） (0.3秒 - 2.0秒)
* **機能:** あなたが話すのをやめた後、AIが文が「完了した」と見なして翻訳に送信するまでの待機時間を決定します。
* **推奨:** 話すのが速い場合は低く保ちます（例：`0.5s`）。文の途中でポーズを取って考える傾向がある場合は、文が半分に切られないように高く設定します（例：`1.0s`またはそれ以上）。

## Max Recording Time（最大録音時間） (5秒 - 30秒)
* **機能:** 翻訳を強制する前にAIが1つのフレーズを録音する時間のハードリミットを設定します。
* **推奨:** `15s`前後に保ちます。これを高すぎると、ポーズをとらずに連続して話すと翻訳に非常に長い遅延が生じる可能性があります。

## AI Accuracy vs Speed [Beam Size]（AIの精度と速度 [ビームサイズ]） (1 - 5)
* **機能:** Faster-Whisperモデルの「ビームサイズ」を調整します。
* **推奨:**
  * 低い値 (1-2): 処理は速いですが、文脈の精度はわずかに低下します。
  * 高い値 (4-5): 高精度ですが、より多くのGPUパワーを必要とし、処理にわずかに時間がかかります。デフォルトは`2`です。
