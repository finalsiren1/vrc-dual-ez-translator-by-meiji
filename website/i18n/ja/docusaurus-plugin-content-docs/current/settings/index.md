# 設定と構成

設定ガイドへようこそ。**Settings（設定）**タブでは、SyncVRCの動作、外観、およびコア処理エンジンをカスタマイズして、好みに完全に合わせることができます。

このセクションでは、以下の内容について説明します:
* **General Settings（一般設定）:** UI言語、ダークモード、テレメトリオプション。
* **AI & Translation Engine（AIと翻訳エンジン）:** AIプロバイダー（Gemini、DeepL、OpenAIなど）の選択、APIキーの管理、およびAI処理デバイス（GPUとCPU）の選択。
* **Shortcut Keys（ショートカットキー）:** 送信（Outgoing）、受信（Incoming）、およびプッシュ・トゥ・トークモード用のカスタムホットキーの設定。

左側のサイドバーから特定のガイドを選択するか、**「Next（次へ）」**をクリックして一般設定の構成を開始してください。
