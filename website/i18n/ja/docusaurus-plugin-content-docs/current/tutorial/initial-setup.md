# 初期設定

SyncVRCの使用を開始する前に、AIと通信システムが正しく機能するように、いくつかの必須設定が必要です。

## 1. APIキーの取得

SyncVRCで翻訳を処理するには、お好みのクラウドAIプロバイダー（Google Gemini、DeepL、OpenAIなど）のAPIキーが必要です。

1. 選択したAIプロバイダーのアカウントを登録します。
2. 開発者またはAPIダッシュボードに移動して、新しいAPIキーを生成します。
3. APIキーをコピーし、安全に保管してください。これをアプリの**Settings（設定）**タブに入力します。

> **推奨:** 無料枠（Geminiの無料枠など）も利用可能ですが、多くの場合、厳密なレート制限があり、翻訳の遅延や一時停止を引き起こす可能性があります。シームレスで途切れのないリアルタイム体験を得るには、プロバイダーの従量課金制（プリペイド）プランへのアップグレードを強くお勧めします。

## 2. NVIDIA CUDA Toolkitのインストール（オプション）

SyncVRCはGPUを利用して高速な音声認識（Faster-Whisper）を実行できます。ただし、これは**オプション**であり、NVIDIA GPU向けに特別に最適化されています。

* **NVIDIAユーザーの場合**: 最大限のパフォーマンスを得るには、[NVIDIA CUDA Toolkit 12.x](https://developer.nvidia.com/cuda-12-9-1-download-archive)をダウンロードしてインストールしてください。**注意:** インストール後にPCを再起動する必要があります。
* **AMD/IntelまたはGPUなしのユーザーの場合**: CUDAをインストールする必要はありません。アプリケーションは自動的にCPUの使用にフォールバックします。
* **デバイスの変更**: GPUとCPUの処理は、いつでも[**Settings > AI & Translation Engine**](../settings/api-engine)で手動で切り替えることができます。

## 3. VRChat OSCの有効化

アプリはOSCプロトコルを介してVRChatと通信し、チャットボックスへの入力とミュートステータスの同期を行います。

1. VRChatを起動し、**ラジアルメニュー**を開きます。
2. **Options > OSC**に移動します。
3. OSCを**Enabled（有効）**に設定します。

**OSCステータスの確認:** 有効にし、VRChatが実行されている状態で、SyncVRCサイドバーの左下を確認してください。正常な接続を確認するために、**OSC Status**インジケーターが**🟢 Detected**に更新されるはずです。

## 4. オーディオデバイスの設定

適切なオーディオルーティングは、翻訳を成功させるための鍵です。

* **Outgoing（送信）**: 物理的なマイクを選択します。
* **Incoming（受信）**: デスクトップ/ゲームのオーディオをキャプチャする**ループバック**または**仮想オーディオデバイス**（VB-Cable、Voicemeeterなど）を選択します。物理的なスピーカーを直接選択することはできません。
