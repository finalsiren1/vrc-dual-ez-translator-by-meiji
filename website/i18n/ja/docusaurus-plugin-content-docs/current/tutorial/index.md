# SyncVRCへようこそ

このドキュメントでは、SyncVRCの基本機能、インストール、および設定について説明します。以下の手順に従って開始してください。

## SyncVRCとは？

SyncVRCは、VRChat専用に設計されたリアルタイム音声翻訳ツールです。高精度な音声認識のためにローカルAI (Faster-Whisper) を活用し、文脈に沿った翻訳のために高度なクラウドAPI (Gemini、DeepL、OpenAI) を利用することで、仮想現実でのシームレスな異言語間コミュニケーションを実現します。

## 主な機能

* 🎙️ **双方向翻訳**: 送信システム（あなたの声をVRChatのチャットボックスに翻訳）と受信システム（他人の声を画面に翻訳）の両方を同時に操作します。
* 🔇 **スマートVRCミュート同期**: VRChatのOSCパラメーターを読み取り、ゲーム内でマイクがミュートされているときに送信翻訳を自動的に一時停止し、プライバシーを確保しAPIクォータを節約します。
* 🔄 **フルオートアップデーター**: ファイルを手動で置き換えることなく、ワンクリックでシームレスなソフトウェアパッチを適用できる自動アップデーターを内蔵しています。
* 🔒 **絶対的なプライバシー**: APIキーはローカルマシン（`config.json`）にのみ保存され、公式のAPIエンドポイントと直接通信します。音声データやキーがリモートで収集または保存されることはありません。
* ⚙️ **プロレベルのオーディオコントロール**: AIの精度（ビームサイズ）、無音タイムアウト、最大録音時間を個人の話すペースに合わせて細かく調整できます。
