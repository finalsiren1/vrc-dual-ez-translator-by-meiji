# 🎧 오디오 설정 및 라우팅

오디오 설정 가이드에 오신 것을 환영합니다. 오디오를 올바르게 라우팅하는 것은 SyncVRC가 사용자와 다른 플레이어의 목소리를 명확하게 "들을" 수 있도록 보장하는 가장 중요한 단계입니다.

이 섹션에서는 송신(Outgoing) 번역을 위한 물리적 마이크와 수신(Incoming) 번역을 위한 가상 오디오 장치를 모두 설정하는 방법을 다룹니다.

왼쪽 사이드바에서 특정 가이드를 선택하거나 **"Next (다음)"**를 클릭하여 송신 마이크 설정을 시작하십시오.
