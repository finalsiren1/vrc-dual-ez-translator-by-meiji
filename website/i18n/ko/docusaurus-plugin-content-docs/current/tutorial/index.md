# SyncVRC에 오신 것을 환영합니다

이 문서에서는 SyncVRC의 기본 기능, 설치 및 설정에 대해 안내합니다. 시작하려면 아래 단계를 따르십시오.

## SyncVRC란?

SyncVRC는 VRChat을 위해 특별히 설계된 실시간 음성 번역 도구입니다. 고정밀 음성 인식을 위해 로컬 AI(Faster-Whisper)를 활용하고 상황에 맞는 번역을 위해 고급 클라우드 API(Gemini, DeepL, OpenAI)를 사용하여 가상 현실에서 원활한 언어 간 커뮤니케이션을 가능하게 합니다.

## 주요 기능

* 🎙️ **양방향 번역**: 송신 시스템(자신의 목소리를 VRChat 채팅창에 번역)과 수신 시스템(다른 사람의 목소리를 화면에 번역)을 동시에 작동합니다.
* 🔇 **스마트 VRC 음소거 동기화**: VRChat OSC 매개변수를 읽어 게임 내에서 마이크가 음소거되었을 때 발신 번역을 자동으로 일시 중지하여 개인 정보를 보호하고 API 할당량을 절약합니다.
* 🔄 **완전 자동 업데이터**: 수동으로 파일을 교체할 필요 없이 원활한 원클릭 소프트웨어 패치를 위한 내장 자동 업데이터가 포함되어 있습니다.
* 🔒 **절대적인 개인정보 보호**: API 키는 로컬 컴퓨터(`config.json`)에만 엄격하게 저장되며 공식 API 엔드포인트와 직접 통신합니다. 음성 데이터나 키는 원격으로 수집되거나 저장되지 않습니다.
* ⚙️ **전문가 수준의 오디오 제어**: AI 정확도(Beam Size), 무음 시간 초과 및 최대 녹음 시간을 개인의 말하는 속도에 맞춰 세밀하게 조정할 수 있습니다.
