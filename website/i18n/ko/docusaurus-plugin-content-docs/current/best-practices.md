# 모범 사례 및 팁

SyncVRC를 최대한 활용하는 것은 기술적인 설정뿐만 아니라 AI와 상호 작용하는 방식에 달려 있습니다. VRChat에서 원활하고 정확하며 빠른 번역을 보장하기 위한 모범 사례는 다음과 같습니다.

## 1. AI를 위한 말하기 (음성 텍스트 변환)
Faster-Whisper 모델은 강력하지만, 여러분의 말을 정확하게 이해하기 위해서는 여전히 선명한 오디오 입력에 의존합니다.
* **명확한 발음:** 웅얼거리거나, 속삭이거나, 너무 작게 말하면 AI가 단어를 잘못 추측하거나 문장 전체를 놓칠 수 있습니다.
* **페이스 및 일시 정지:** 클라우드 API는 텍스트를 문장 단위로 번역합니다. 끊임없이 길게 이어지는 문장으로 말하면 AI는 사용자가 말을 멈출 때까지 기다렸다가 오디오를 번역으로 보냅니다. 자연스럽게 숨을 쉬고 문장 끝에서 살짝 쉬어주면 번역을 더 빨리 시작할 수 있습니다.
* **배경 소음 피하기:** AI가 일부 소음을 필터링하지만, 시끄러운 배경 음악, 책상을 두드리는 소리 또는 물리적 마이크에 직접 부는 선풍기 바람은 인식 정확도를 떨어뜨립니다.

## 2. 푸시 투 토크 (Push-to-Talk) 워크플로우 최적화
무료 등급 API를 사용하거나 시끄러운 퍼블릭 인스턴스에서 플레이하는 경우 **누르고 녹음하기 (Push to Record)** 모드가 매우 유용합니다.
* 버튼을 길게 누르고, 생각을 끝까지 말한 다음 놓습니다. 이렇게 하면 AI가 처리할 수 있도록 완벽하게 패키징된 오디오 클립이 제공되어 API 할당량을 절약하고 AI가 배경 대화를 번역하여 발생하는 당황스러운 "핫 마이크" 순간을 방지합니다.

## 3. VRChat 은어 및 컨텍스트 처리
* **표준어 유지:** 클라우드 AI 모델은 표준 언어로 학습됩니다. 과도한 인터넷 은어, VRChat 특유의 용어 또는 혼합 언어 문장 (예: "Japanglish")은 번역기를 혼란스럽게 할 수 있습니다.
* **단순함 유지:** 확실하지 않을 때는 더 간단한 문장 구조를 사용하십시오. "다른 월드로 갈게요"라는 표현은 "저 빠지고 포털 탈게요"보다 훨씬 안정적으로 번역됩니다.

## 4. VRChat 에티켓
* **기대치 설정:** VRChat 바이오 또는 상태 표시 버블에 "(번역기 사용)" 또는 "(AI 음성)"을 넣는 것을 강력히 권장합니다. 이를 통해 다른 플레이어는 답변에 1~2초의 지연이 있을 수 있다는 것을 예상하게 되어 대화가 훨씬 자연스럽게 진행됩니다.
