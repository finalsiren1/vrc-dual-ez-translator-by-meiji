# DeepL API 키 얻기

SyncVRC는 **DeepL API**를 통한 번역을 지원합니다. 이 서비스는 특히 유럽 및 아시아 언어에 대해 자연스럽게 들리는 번역으로 잘 알려져 있습니다.

:::warning 면책 조항
개발자가 일일 테스트에 DeepL API를 주로 사용하지 않으므로 이 튜토리얼이 불완전하거나 약간 오래되었을 수 있습니다. 불일치하는 사항이 있는 경우 공식 DeepL 설명서를 참조하십시오.
:::

---

## 키를 얻는 단계

1. **DeepL 계정 만들기:** [DeepL Pro](https://www.deepl.com/pro-api) 웹사이트로 이동합니다.
2. **API 요금제 선택:** **"DeepL API Free"**(월 최대 500,000자) 또는 **"DeepL API Pro"** 요금제를 선택합니다.
    * *참고: 무료 요금제의 경우에도 DeepL은 일반적으로 신원 확인을 위해 유효한 신용 카드가 필요합니다.*
3. **계정 콘솔에 액세스:** 로그인한 후 **"계정"** 탭으로 이동합니다.
4. **인증 키 찾기:** 계정 페이지 맨 아래로 스크롤하여 **"DeepL API용 인증 키"**를 찾습니다.
5. **복사 및 붙여넣기:** 키를 복사하여 SyncVRC의 **Settings (설정)** 탭 내의 DeepL API 섹션에 붙여넣습니다.

---

## DeepL API Free vs. Pro

* **DeepL API Free:** 무료로 넉넉한 월 문자 수 제한을 제공하지만, 대부분의 지역에서 신용 카드 확인이 필요합니다.
* **DeepL API Pro:** 문자 수 제한을 없애고(종량제) 최대한의 데이터 보안을 제공하여 텍스트가 저장되거나 모델을 학습하는 데 절대 사용되지 않도록 보장합니다.
