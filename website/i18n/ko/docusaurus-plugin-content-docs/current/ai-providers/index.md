# 클라우드 AI 제공업체 및 API 키

클라우드 AI 섹션에 오신 것을 환영합니다. SyncVRC에서는 VR 게임 성능에 영향을 주지 않으면서 실시간으로 매우 정확한 번역을 제공하기 위해 하이브리드 AI 접근 방식을 활용합니다.

## SyncVRC의 AI 활용 방법

번역 파이프라인은 두 가지 전문 작업으로 나뉩니다.
1. **음성 텍스트 변환 (로컬):** 완전히 로컬 컴퓨터에서 실행되는 **Faster-Whisper** 모델을 사용하여 음성을 텍스트로 변환합니다. 이를 통해 즉각적인 음성 인식이 보장됩니다.
2. **컨텍스트 번역 (클라우드):** 음성이 텍스트로 변환되면 텍스트가 고급 **클라우드 AI API**(예: Google Gemini, DeepL 또는 OpenAI)로 전송되어 대상 언어로 번역됩니다.

## 로컬 번역 모델을 사용하지 않는 이유

SyncVRC가 Faster-Whisper와 함께 번역 언어 모델을 로컬로 실행하지 않는 이유가 궁금할 수 있습니다. 대답은 **시스템 성능**입니다.

자연스럽고 상황을 인식하는 번역이 가능한 고품질 대규모 언어 모델(LLM)을 실행하려면 엄청난 양의 컴퓨팅 파워와 비디오 RAM(VRAM)이 필요합니다. VRChat과 같은 요구 사항이 많은 VR 게임 및 Faster-Whisper 모델과 동시에 무거운 로컬 LLM을 실행하면 PC에 심각한 병목 현상이 발생합니다. 이로 인해 심각한 프레임 드롭, 렉이 발생하여 VR을 플레이할 수 없게 됩니다.

텍스트 번역 단계를 클라우드 API로 오프로드함으로써 SyncVRC는 다음을 보장합니다.
* **게임에 미치는 영향 제로:** PC의 리소스는 VRChat을 원활하게 실행하는 데 전적으로 사용됩니다.
* **초고속:** 작은 텍스트 문자열을 클라우드 서버로 보내는 데는 몇 밀리초밖에 걸리지 않습니다.
* **뛰어난 정확도:** 클라우드 API는 경량의 로컬 모델보다 은어, 문맥 및 문법을 훨씬 더 잘 이해하는 수십억 개의 매개변수를 가진 대규모 모델을 활용합니다.

## API 키 얻기

SyncVRC를 이러한 클라우드 서비스에 연결하려면 선택한 AI 제공업체의 개인 API 키를 제공해야 합니다.

왼쪽 사이드바에서 제공업체를 선택하거나 **"Next (다음)"**를 클릭하여 **Google Gemini API 키**를 등록하고 얻는 방법을 알아보십시오.
