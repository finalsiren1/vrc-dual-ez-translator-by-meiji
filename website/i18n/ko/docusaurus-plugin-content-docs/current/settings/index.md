# 설정 및 구성

설정 가이드에 오신 것을 환영합니다. **Settings (설정)** 탭에서는 SyncVRC의 동작, 모양 및 핵심 처리 엔진을 사용자의 선호에 완벽하게 맞게 사용자 지정할 수 있습니다.

이 섹션에서는 다음 내용을 다룹니다.
* **General Settings (일반 설정):** UI 언어, 다크 모드 및 원격 분석 옵션.
* **AI & Translation Engine (AI 및 번역 엔진):** AI 제공업체(예: Gemini, DeepL, OpenAI) 선택, API 키 관리 및 AI 처리 장치(GPU vs CPU) 선택.
* **Shortcut Keys (단축키):** 송신(Outgoing), 수신(Incoming) 및 푸시 투 토크 모드에 대한 사용자 지정 단축키 구성.

왼쪽 사이드바에서 특정 가이드를 선택하거나 **"Next (다음)"**를 클릭하여 일반 설정을 시작하십시오.
