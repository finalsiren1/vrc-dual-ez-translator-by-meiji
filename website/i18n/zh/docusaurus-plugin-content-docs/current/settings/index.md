# 设置与配置

欢迎来到设置指南。**Settings (设置)** 选项卡允许您自定义 SyncVRC 的行为、外观和核心处理引擎，以完美满足您的偏好。

在本节中，我们将介绍：
* **General Settings (常规设置):** UI 语言、深色模式和遥测选项。
* **AI & Translation Engine (AI 与翻译引擎):** 选择您的 AI 提供商（例如 Gemini、DeepL、OpenAI）、管理 API 密钥以及选择您的 AI 处理设备（GPU 与 CPU）。
* **Shortcut Keys (快捷键):** 为发送 (Outgoing)、接收 (Incoming) 和按键说话 (Push-to-Talk) 模式配置自定义热键。

请从左侧边栏中选择特定的指南，或点击**“Next (下一步)”**开始配置您的常规设置。
