# 欢迎使用 SyncVRC

本文档将引导您了解 SyncVRC 的基本功能、安装和设置。请按照以下步骤开始。

## 什么是 SyncVRC？

SyncVRC 是一款专为 VRChat 设计的实时语音翻译工具。它利用本地 AI (Faster-Whisper) 进行高精度的语音识别，并利用高级云 API (Gemini, DeepL, OpenAI) 进行上下文翻译，从而实现在虚拟现实中的无缝跨语言交流。

## 主要功能

* 🎙️ **双向翻译**: 同时运行发送系统（将您的声音翻译到 VRChat 聊天框）和接收系统（将他人的声音翻译到您的屏幕上）。
* 🔇 **智能 VRC 静音同步**: 读取 VRChat OSC 参数，在您在游戏中静音麦克风时自动暂停发送翻译，从而确保隐私并节省 API 额度。
* 🔄 **全自动更新器**: 内置自动更新器，实现无缝的一键式软件修补，无需手动替换文件。
* 🔒 **绝对隐私**: API 密钥严格存储在您的本地计算机 (`config.json`) 上，并直接与官方 API 端点通信。不会远程收集或存储任何语音数据或密钥。
* ⚙️ **专业级音频控制**: 提供对 AI 精度（Beam Size）、静音超时和最大录音时长的精细调整，以匹配您个人的语速。
