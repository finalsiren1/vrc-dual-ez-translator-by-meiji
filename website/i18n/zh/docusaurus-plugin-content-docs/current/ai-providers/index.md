# 云 AI 提供商与 API 密钥

欢迎来到云 AI 部分。在 SyncVRC 中，我们利用混合 AI 方法来提供实时、高精度的翻译，而不会影响您的 VR 游戏性能。

## SyncVRC 如何使用 AI

翻译流程分为两个专门的任务：
1. **语音转文本 (本地):** 我们使用完全在您的本地机器上运行的 **Faster-Whisper** 模型将您的声音转换为文本。这确保了即时的语音识别。
2. **上下文翻译 (云端):** 一旦您的语音被转录，文本将被发送到高级的**云 AI API**（例如 Google Gemini、DeepL 或 OpenAI）以翻译成目标语言。

## 为什么不使用本地翻译模型？

您可能想知道为什么 SyncVRC 不在本地与 Faster-Whisper 一起运行翻译语言模型。答案是**系统性能**。

运行能够进行自然、具有上下文感知能力的翻译的高质量大型语言模型 (LLM) 需要大量的计算能力和显存 (VRAM)。在运行 VRChat 这样要求苛刻的 VR 游戏和 Faster-Whisper 模型的同时，再运行沉重的本地 LLM 将严重成为您电脑的瓶颈。这会导致大量掉帧、卡顿和无法游玩的 VR 体验。

通过将文本翻译步骤卸载到云 API，SyncVRC 保证：
* **对游戏零影响:** 您的电脑资源完全专用于流畅地运行 VRChat。
* **闪电般的速度:** 将少量文本字符串发送到云服务器只需几毫秒。
* **卓越的准确性:** 云 API 利用庞大的、数十亿参数的模型，它们比任何轻量级的本地模型都能更好地理解俚语、上下文和语法。

## 获取您的 API 密钥

要将 SyncVRC 连接到这些云服务，您必须提供您选择的 AI 提供商的个人 API 密钥。

请从左侧的侧边栏中选择一个提供商，或点击**“Next (下一步)”**了解如何注册并获取 **Google Gemini API 密钥**。
